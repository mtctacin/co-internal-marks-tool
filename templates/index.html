Internal / CCA Form Preparation Tool for OBE Based Courses

This tool assists in preparing Internal / Continuous Comprehensive Assessment (CCA) forms for courses that contain multiple Course Outcomes (COs) and several formative or summative assessment components mapped to those COs.

In practice, many assessments are conducted for higher raw marks (for example, a test conducted for 30 marks), while the CCA format may require the marks to be entered in a smaller normalized scale (for example, 4 marks).

This tool automatically:

• Normalizes marks from the original assessment scale to the required CCA scale
• Handles multiple COs and multiple components per CO
• Processes student marks in bulk
• Generates the final normalized marks in a format ready for entry into the CCA form

The output can be used directly while preparing the final CCA statement, reducing manual calculation errors and saving time.
